## 🔥 Quick Commands

### Start Streamlit (Already Running!)
```bash
streamlit run streamlit_app.py
```

### Run with Helper Script
```bash
# Mac/Linux
./run_streamlit.sh

# Windows
run_streamlit.bat
```

### Deploy Full Stack with Docker
```bash
docker-compose -f docker-compose.streamlit.yml up
```

### Stop Streamlit
```
Press Ctrl+C in terminal
```

### Open Different Port (if 8501 is busy)
```bash
streamlit run streamlit_app.py --server.port=8502
```

## 📞 Need Help?

| Problem | Solution |
|---------|----------|
| **App won't load** | Make sure port 8501 is free |
| **Port busy** | Use: `streamlit run streamlit_app.py --server.port=8502` |
| **API errors** | Start backend: `cd backend && npm start` |
| **Module not found** | Run: `pip install -r requirements_streamlit.txt` |
| **Clear issues** | Remove: `rm -rf ~/.streamlit` |
